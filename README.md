# 🎯 Adivina el Número Mágico

## 📌 Descripción

Este proyecto consiste en una práctica de DOM y BOM, en la cual se implementa un pequeño juego interactivo utilizando **JavaScript** y **Bootstrap**.

El objetivo del juego es que el usuario adivine un **número aleatorio** generado por el sistema. A través de una interfaz simple, el usuario puede ingresar su intento y recibir retroalimentación inmediata mediante `alert()`.

---

## 🧠 Enunciado del ejercicio

> Crea una web con Bootstrap y JS, que contenga un botón "Comenzar el juego". En ese momento se crea un número aleatorio que el usuario deberá adivinar.
>
> La interfaz del usuario debe tener además:
> - Un input para ingresar un número
> - Un botón para enviar
>
> Al presionar el botón "Enviar":
> - Mostrar con un `alert()` si el usuario adivinó o no el número mágico.
> - Si no lo adivinó, indicarle con un `alert()` si el número que ingresó es mayor o menor al número mágico.
>
> Cuando el usuario adivine el número, mostrar un mensaje indicando que adivinó correctamente.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (DOM y BOM)

---

## 🎮 ¿Cómo funciona?

1. Al presionar el botón **"Comenzar el juego"**, se genera un número aleatorio del 1 al 100.
2. El usuario ingresa un número en el campo y presiona **"Comprobar resultado"**.
3. El sistema compara el número ingresado con el número mágico:
   - Si lo adivina: se muestra un mensaje de felicitación.
   - Si no lo adivina: se muestra un `alert` indicando si debe probar un número mayor o menor.
4. El botón "Intentar de nuevo" permite reiniciar el juego con un nuevo número.
